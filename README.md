# 🎮 Zolo Quiz

Welcome to Zolo quiz — a modern web app for playing and creating quizzes.
Zolo stand for "You only live one", reminding us to keep learning and improving every day.

This is where you can browse quiz categories, play quizzes, track your results, and (with an account) create and manage your own quizzes.

## 👋 About

Quiz Game is a full-stack quiz application built with Next.js. It focuses on a smooth playing experience, clear results, and optional user accounts for saving progress and creating quizzes.

## 🚀 Tech Stack

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Supabase** (auth & database)
- **TanStack Query**
- **Framer Motion**
- **Radix UI**
- **Zustand**
- **next-intl**

## 📂 Features

- Quiz categories and browsing
- Play quizzes with multiple question types
- Results and score tracking
- User authentication (login / sign up)
- Create and manage your own quizzes
- Responsive layout
- Animations and confetti on success
